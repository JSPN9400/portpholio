ℹ️ How to Use Admin Panel
📌 Projects — Add/edit/delete projects. Mark one as "Featured" to show on homepage.
📌 Blog Posts — Write and manage articles. Latest one shows on homepage automatically.
📌 About Me — Update your bio, tagline, and profile description.
📌 Contact Info — Change phone, email, LinkedIn, GitHub links.
📌 Resume — Update resume link or upload instructions.
📌 Save All Changes — Click the blue Save button. Changes saved in browser storage.
⚠️ Data is saved in your browser's LocalStorage. After saving, copy the generated JSON from Settings → Export and paste into your JSON files to make changes permanent on GitHub.
